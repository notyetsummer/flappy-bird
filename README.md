# Flappy Bird (улучшенная версия)

Игра на **Python + pygame**. Запуск из корня проекта:

```bash
python3 -m pip install -r requirements.txt
python3 "Export_Version_2 (2).py"
```

В имя файла есть пробелы и скобки — в командной строке удобнее брать его в кавычках.

Локальный файл **Results.txt** (таблица очков) в репозиторий не коммитится — он указан в `.gitignore`.

## Зависимости

См. `requirements.txt`: `pygame`, `pygame-menu`.
